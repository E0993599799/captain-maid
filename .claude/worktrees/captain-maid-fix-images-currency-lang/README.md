# Captain Maid 2.0

Modern website for Captain Maid premium home cleaning products.

## Tech Stack

- Next.js 15 + React 19 + TypeScript
- Tailwind CSS
- next-intl (Thai/English)
- Vercel deployment

## Features

✅ Multi-language support (Thai/English)
✅ SEO optimized
✅ Mobile-responsive
✅ Product showcase
✅ Blog section
✅ Contact form
✅ Fast performance

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build & Deploy

```bash
npm run build
vercel
```

## Project Structure

```
app/[locale]/
├── page.tsx          # Home
├── products/         # Products page
├── blog/             # Blog
├── about/            # About page
└── contact/          # Contact page
i18n/
locales/
middleware.ts
```
