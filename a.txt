# Web Visual Design Skill

## Mission
Create beautiful, credible, conversion-focused web pages using the brand's real assets and business goal.

## Design Principles
- Clear hierarchy before decoration
- Premium spacing, not crowded UI
- One dominant CTA per section
- Real product imagery over generic mockups
- Mobile-first section rhythm
- Motion must support meaning, not distract

## Layout Rules
- Hero must include: headline, subheadline, CTA, trust cue, visual anchor
- Every section needs one job only
- Alternate dense and breathable sections
- Use card grids only when comparison is useful
- Keep max content width readable

## Visual Rules
- Use brand color as accent, not background everywhere
- Use 8px spacing system
- Use consistent radius and shadow scale
- Avoid random gradients
- Avoid too many font weights
- Use real images from assets folder first

## Motion Rules
- Initial hero fade/slide only
- Cards stagger on viewport
- CTA hover: subtle lift
- No infinite animation unless decorative and low opacity
- Respect reduced-motion

## Quality Gate
Before final answer:
- Check mobile layout
- Check CTA visibility
- Check contrast
- Check image usage
- Check section rhythm
- Check no placeholder content
